# Projet MNIST avec Keras

Ce projet implémente un réseau de neurones fully connected pour classifier les chiffres manuscrits du dataset MNIST.

## Contenu du projet
- Chargement et normalisation du dataset MNIST
- Construction d’un réseau dense (Dense + Dropout)
- Entraînement du modèle
- Évaluation sur les données de test
- Sauvegarde du modèle au format `.h5`

python train_model.py
